THIS IS A CONSOLE BASED BANK MANAGEMENT SYSTEM
BY TAMANWIT SADHUKHAN

-- THIS PROJECT CONTAINS AN ADMIN PROFIE & CUSTOMER PROFILE
-- IT IS A DEDICATED DATABASE WHICH GETS UPDATED ON EVERY SINGLE TRANSACTION AND ACCOUNT CREATION AND DOESNOT GET DELETED ON RESTARTING THE PROGRAM.
-- LOAN/LOCKER APPLICATION CAN BE SUBMITTED BY CUSTOMER AND HAS TO BE APPROVED/REJECTED BY ADMIN
-- IT CAN GENERATE MINI STATEMENT (TRANSACTION HISTORY) OF ANY USER
-- IT DOESNOT REQUIRE ANY INTERNET CONNECTION FOR THE PROGRAM TO RUN AND EVEN THE DATABASE TO FUNCTION PROPERLY
